# Game of Life (PHP)

This project is a simple implementation of **Game of Life**
using PHP. The simulation runs on a **25 x 25 grid** and demonstrates
the **Glider pattern**.

For now Assume to run 20 time generation

## Requirements

-   PHP installed on your system
-   Command Line / Terminal

## How to Run

1.  Open Terminal or Command Prompt.
2.  Navigate to the project folder.

Example:

cd path/to/project

3.  Run the PHP program:

**php game_of_life.php**

4.  The program will print the grid in the console and update it every
    generation (tick).

## Output Symbols

0 = Alive Cell\
. = Dead Cell
